
🚀 JudgeNest — Online Coding Judge Platform

JudgeNest is a full-stack coding platform inspired by LeetCode where users can solve problems, write code, run it, and compete on a leaderboard.

---

## 🔥 Features

- Monaco Code Editor (VS Code-like experience)
- Run and Submit Code functionality
- Test case based evaluation system
- Dynamic leaderboard ranking
- User authentication system
- Profile management
- Clean and modern UI

---

## 🧠 Tech Stack

Frontend:
- React.js
- TypeScript
- Custom CSS / UI design
- Monaco Editor

Backend:
- Node.js
- Express.js
- MongoDB

---

## ⚙️ How It Works

1. User logs in or signs up  
2. Selects a coding problem  
3. Writes code in the editor  
4. Runs code to check output  
5. Submits solution for evaluation  
6. Backend checks using test cases  
7. Leaderboard updates based on solved problems
